## 📦 Setup Instructions

1. **Download all files**
   - index.html
   - index.css
   - index.js
   - vignesh_Resume.pdf (you need to add your resume)

2. **Add your resume**
   - Place your resume PDF in the same folder
   - Name it `vignesh_Resume.pdf`

3. **Update Resume Link (Optional)**
   - If you want to link to Google Drive instead:
   - Open `index.js`
   - Replace the URL in the `resume()` function with your Google Drive link

4. **Open the website**
   - Simply open `index.html` in your browser
   - Or use a local server for better performance

## 🌐 Deployment

You can deploy this portfolio to:
- **GitHub Pages**: Free and easy
- **Vercel**: Fast and reliable
- **Netlify**: Simple deployment
- **Any static hosting service**

### GitHub Pages Deployment:
1. Create a new repository named `yourusername.github.io`
2. Upload all files to the repository
3. Enable GitHub Pages in Settings
4. Your site will be live at `https://yourusername.github.io`
